# NeetCode practices

## About
- This repo consists my neetcode.io (the more refined version of the leetcode) exercises.

- Preferred to use the IDE for instant feedbacks and cloud-saving opportunity. Also the agent can help when stuck which is so convenient.

## Why C++?
- Because we have absolute control on the hardware with C++, thanks to zero cost abstractions. In python on the other hand, you create a virtual layer on top of the abstracted built-in structures. Which is not the thing I want.
